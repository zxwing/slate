Simple API

A simple API to learn how to write OpenAPI Specification

Version: 1.0.0

/persons

GET

Summary: Gets some persons

Description: Returns a list containing all persons. The list supports paging.

Parameters

Name Located in Description Required Schema
pageSize query Number of persons returned No integer
pageNumber query Page number No integer

Responses

Code Description Schema
200 A list of Person
POST

Summary: Creates a person

Description: Adds a new person to the persons list.

Parameters

Name Located in Description Required Schema
person body The person to create. No

Responses

Code Description
204 Persons succesfully created.
400 Persons couldn't have been created.

/persons/{username}

GET

Summary: Gets a person

Description: Returns a single person for its username.

Parameters

Name Located in Description Required Schema
username path The person's username Yes string

Responses

Code Description Schema
200 A Person
404 The Person does not exists.
